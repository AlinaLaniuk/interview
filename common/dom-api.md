# Методы вставки узлов 

Вставка:

- append - в конец ноды
- prepend - в начало ноды
- before - до ноды
- after - после ноды
- replaceWith - заменить ноду

Методы могут вставлять сразу несколько нод или строк кода (нужно передать через запятую). Вставляют как текст
